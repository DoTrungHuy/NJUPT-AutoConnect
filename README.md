# NJUPT-AutoConnect

南京邮电大学校园网手动一键认证工具。第一次填账号、密码、运营商，之后打开页面点一次“连接校园网”即可。

这个项目不提供账号共享、不收集密码、不做远程代登。账号和密码只保存在你自己的设备上。

## 最快使用

1. 打开 `web/index.html`。
2. 输入账号、密码、运营商，按需勾选“记住密码”。
3. 以后连上南邮 Wi-Fi 后，打开这个页面点“连接校园网”。

手机上可以把页面添加到主屏幕，之后就像一个小工具一样打开使用。

## GitHub Pages

仓库发布到 GitHub 后，内置 GitHub Actions 会把 `web/` 目录发布到 GitHub Pages。用户提前打开一次页面后，Service Worker 会缓存页面，后面校园网未认证时也能打开工具界面。

如果没有开启 Pages，也可以直接下载仓库后打开 `web/index.html`。本地文件模式下不能注册 Service Worker，但一键认证功能仍可用。

## Windows 双击脚本

Windows 用户也可以用脚本版：

```text
scripts/windows/connect.cmd
```

第一次双击会提示账号、密码、运营商。密码用 Windows DPAPI 加密保存到当前用户目录，后续再次双击会直接尝试认证。

脚本版不创建任务计划、不后台运行、不自启动，只做主动双击一键联网。

## 运营商

```text
教育网: njupt
电信:   ctcc / @njxy
移动:   cmcc / @cmcc
```

如果不确定，优先尝试电信 `ctcc`。

## 隐私说明

- PWA 的“记住密码”使用浏览器本地存储，只适合个人设备。
- 如果不想保存密码，可以关闭“记住密码”，以后每次只输入密码再点连接。
- Windows 脚本使用当前 Windows 用户的 DPAPI 加密密码。
- 项目没有服务器，不上传账号或密码。

## 故障处理

- 点“连接校园网”失败时，页面会显示“打开登录链接”备用按钮。
- 如果浏览器阻止跨域脚本或校园网证书异常，请使用备用按钮。
- 如果已经能正常上网，就不要反复点击连接，避免校园网 Portal 重复登录。
